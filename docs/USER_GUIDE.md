# 📖 NaviDash 用户指南 (User Guide)

欢迎使用 NaviDash！本指南将帮助您快速上手，打造属于您的个性化起始页。

---

## 🎨 界面概览

NaviDash 的主界面由三个部分组成：
1.  **中央仪表盘 (Main Dashboard)**：放置各种小组件（时钟、天气、便签等）。
2.  **左侧侧边栏 (Sidebar)**：鼠标悬停在左侧边缘即可唤出，用于管理您的书签和导航。
3.  **顶部/底部工具栏**：包含搜索框、编辑模式开关和全局设置。

---

## 🧩 小组件管理 (Widgets)

### 添加组件
1.  点击页面底部的 **编辑模式** 图标（通常是一个解锁🔓图标，或者直接点击“+”号）。
2.  在弹出的组件库中选择您喜欢的组件（如“时钟”、“天气”、“待办事项”）。
3.  组件会自动添加到仪表盘底部的空闲位置。

### 移动与调整大小
1.  进入 **编辑模式**。
2.  **移动**：按住组件左上角的 **拖拽手柄**，将其拖动到任意位置。
3.  **调整大小**：拖动组件右下角的 **缩放手柄**，调整其长宽比例。

### 配置组件
在编辑模式下，点击组件右上角的 **设置 (⚙️)** 按钮即可进行个性化配置。

#### 🕰️ 时钟组件 (Clock)
- **样式切换**：在设置中选择 **Digital (数字)** 或 **Analog (拟真)**。
  - *提示：拟真时钟采用了无边框设计，配合洞洞板背景效果极佳。*

#### ☀️ 天气组件 (Weather)
- **位置设置**：输入您所在的城市名称（支持拼音或英文）。
- **显示内容**：自动显示当前温度、天气状况、体感温度等。

#### 📝 便签与待办 (Memo & Todo)
- **便签**：直接点击便签区域即可输入内容，支持自动保存。
- **待办**：点击“+”添加任务，点击复选框标记完成。

## 🖼️ 背景与视觉定制

点击侧边栏底部或主界面上的 **设置 (Settings)** 按钮进入全局设置。

### 默认背景 (洞洞板)
NaviDash 默认使用经典的 **洞洞板 (Pegboard)** 风格，提供整洁、实体的质感。

### 自定义壁纸
1.  在设置面板中找到 **背景图片 URL** 输入框。
2.  粘贴任意网络图片的链接（以 `http` 或 `https` 开头）。
    - *推荐来源：Unsplash, Bing Daily Wallpaper 等。*
3.  **视觉微调**：
    - **模糊度 (Blur)**：拖动滑块使背景模糊，让前景组件更突出。
    - **遮罩透明度 (Opacity)**：增加一层黑色遮罩，提高文字在亮色背景上的可读性。

## 📑 书签管理 (Sidebar)

### 快速导航
- 鼠标移至屏幕左侧边缘，侧边栏会自动滑出。
- 点击分类文件夹展开/折叠，点击链接直接跳转。

### 编辑书签
1.  点击侧边栏底部的 **管理 (Manage)** 按钮。
2.  **添加**：点击“新建分类”或“新建链接”。
3.  **排序**：直接拖拽书签或文件夹进行重新排序。
4.  **图标**：输入图标名称（如 `github`, `youtube`, `code`）即可自动匹配图标。

## 💾 数据备份与恢复

NaviDash 重视您的数据安全，所有配置均存储在本地。

### 导出备份
1.  打开 **设置** 面板。
2.  点击 **导出数据 (Export)** 按钮。
3.  系统会生成一个 `.json` 文件，包含您所有的布局、组件配置和书签数据。

### 恢复数据
1.  打开 **设置** 面板。
2.  点击 **导入数据 (Import)** 按钮。
3.  选择您之前备份的 `.json` 文件，确认后页面将自动刷新并应用配置。
4.  *注意：导入操作会覆盖当前的配置，请谨慎操作。*

## ❓ 常见问题 (FAQ)

**Q: 为什么我的天气不显示？**
A: 请检查网络连接，并确保输入的城市名称正确。部分地区可能需要更精确的名称（如 "Beijing, China"）。

**Q: 如何重置所有设置？**
A: 在设置面板底部有一个红色的 **重置 (Reset)** 按钮，点击后将恢复到初始状态（注意：此操作不可撤销）。

**Q: 拟真时钟为什么没有背景框？**
A: 这是为了模拟真实的挂钟效果，让其自然地“挂”在背景墙上。如果您切换回数字时钟，将会看到标准的卡片背景。
